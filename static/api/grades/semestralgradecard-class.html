SemestralGradeCard

SemestralGradeCard class Null safety

Interface to display only CourseGrade.course.subject and CourseGrade.semestralGrade from a CourseGrade.

Its background is green if CourseGrade.isApproved is true, otherwise is AppColors.gray.shade100.
